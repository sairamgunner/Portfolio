Projects

Coffee Cup Business Analysis using Tableau

A fictitious coffee shop which has stores all over Canada has been collecting data to understand and interpret valuable trends, and to empower them with insights from the data that may or may not have been visible without proper visualizations. In this project, the given data had been cleaned using MS Excel and OpenRefine and beautiful visuals have been created using Tableau. The visuals are dynamic to an extent that during a meeting, the stakeholders are facilitated with the power to drilldown from province level to each store level and gather interesting insights from the data.

Willis Towers Watson

As an Associate Consultant at Capgemini, I had the opportunity to work with Willis Towers Watson as clients. As a B.A., I have helped 3 projects with gap analysis, requirement gathering and elicitation to successful on-time go-live of each web application followed by bi-weekly iterative improvements to the applications with proper backlog management using Azure DevOps. The domains of these projects were Insurance, Capital Markets and Pensions. In these projects I cherished the chance to work with client teams, business users, leadership teams and technical teams from the USA, Canada, the UK, Vietnam and the Philipines.

Toronto House Pricing

This project mainly aims at collecting data from a real estate broking website called Zoocasa by scraping the website using Selenium with Python. All steps including navigating to the website through Chrome, logging in using credentials, checking the different houses listed, collecting the data from the website into a pandas dataframe, and printing the collected data into a .csv file has all been automated.
